# java-lotto
로또 미션 진행을 위한 저장소

## 기능 요구 사항
* 로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급해야 한다.
* 로또 1장의 가격은 1000원이다.

## 프로그래밍 요구 사항
* indent의 depth를 1단계로
* else 사용 금지
* 메소드의 크기가 최대 10라인을 넘지 않도록 구현한다.
* 배열 대신 ArrayList를 사용한다.
* enum을 적용해 구현한다.

## 구현할 기능 목록

    * [x] **예외** 한 장도 살수 없는 금액인 경우
    
* [x] 금액으로 로또 장수를 결정하는 기능

* [x] 로또를 섞는 기능

* [x] 공 6개를 꺼내는 기능

* [x] 구매한 로또들을 출력하는 기능

* [x] 지난 주 당첨 번호를 입력받는 기능(구분자는 콤마(",")로 구분)
